# 树
###### 思路：
1、层序遍历
2、找到出口
###### 题型：
1、前中后序遍历、递归、非递归
2、DFS、BFS，深度优先，广度优先
3、层序遍历
# 数组
- ###### 题型1、最长递增子序列
思路：类别同样长的序列，不断更新让前面的更小，更能容纳。

- ###### 题型2、最长重复子数组
思路： 将两个数组作为方格盘的横竖两个列，那么，重复的子数组会形成一个斜杠。使用dp动态规划。
